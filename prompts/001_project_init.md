# Tasca
Crea la configuració inicial al repository `uoc-vd-practica-part2p per a una aplicació web per a un projecte de visualització de dades.

Consultar l'arxiu README.md per a més detalls sobre el projecte i les dades que s'utilitzaran.
Llegeix el fitxer de dades per entendre la seva estructura i com es pot utilitzar per a la visualització.

# Scope
- Configuració inicial del projecte amb React.
- Integració de D3.js per a la visualització de dades.
- Creació d'una estructura de carpetes clara i organitzada.
- Configuració de les dependències necessàries.
- Llegeix el fitxer de dades per entendre la seva estructura i com es pot utilitzar per a la visualització.
- Per començar, només cal crear una pàgina d'inici amb un gràfic de mostra utilitzant D3.js.
